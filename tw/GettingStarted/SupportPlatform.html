支援的平臺

Kreogist Cuties 允許您在多個系統平臺上運行。

開發平臺

Kreogist Cuties 為下列作業系統提供編譯好的二進位安裝包：

  • Microsoft® Windows®

  • Ubuntu Linux

  • Mac OS X

    參考設置

    一套設置是指對於某個平臺來說，一套編譯器、調試器和其他工具的搭配組合。下面是一些 Kreogist Dev Team 已進行過測試的設置，以供參考。

    作業系統平臺編譯器
    Ubuntu Linux 13.04, X11（64位）Ubuntu 提供的編譯器
    Microsoft Windows 7（32位和64位）MinGW-builds gcc 4.8.1（32位）
    Microsoft Windows 8（32位和64位）MinGW-builds gcc 4.8.1（32位）
    Mac OS X 10.8 "Mountain Lion", Cocoa（64位）Apple 提供的 GCC 編譯器

    您也可以使用您自己的設置編寫代碼。瞭解更多資訊，請參閱編譯器設置。

    可供使用的二進位安裝包

    Cuties 為所支援的作業系統提供了二進位安裝包。您可以從 http://kreogist.github.io/Cuties/cn/downloads.html 獲得。此處提供了所有 Cuties 所支援作業系統的下載連結。
