快速入门教程

这一章节通过编写一个最基本的 C 语言代码向您描述了如何使用 Cuties 去创建您的代码。您也可以通过这个实例检测您的 Cuties 是否成功安装。

创建一个新的文件

从欢迎屏幕，或者从主菜单（Windows/Linux）/菜单栏（Mac OS X）、工具栏创建一个新的源代码文件。如需了解更多有关创建文件的内容，请参阅创建新的代码文件。

编写你的代码

在代码编辑器中编写你的代码，这里我们提供一个简单的例子：

#include <stdio.h>
int main(int argc, char *argv[])
{
	printf("Hello World!\n");
	return 0;
}

这是一个非常简单的C语言代码，它的作用是在屏幕上显示“Hello World!”。你可以复制或者键入这段代码到 Cuties 之中。将其保存成 C 源代码文件。如需了解更多有关保存源代码文件的内容，请参阅保存你的代码。

编译和运行

你可以手动编译这个源代码然后手动运行。现在你还可以通过单击工具栏中的“编译运行”按钮直接完成这项操作。如需了解更多有关编译运行的内容，请参阅构建你的程序。

现在屏幕上会出现 Cuties 的命令行运行程序（一个命令行窗口），上面显示四行内容：第一行内容是“Hello World!”，第二行是一个空行。第三行显示了程序的运行时间，光标处于第四行，提示您请按下键盘上的任意一个按键（Windows）/回车键（其他系统）来关闭命令行运行程序。
