快速入門教程

這一章節通過編寫一個最基本的 C 語言代碼向您描述了如何使用 Cuties 去創建您的代碼。您也可以通過這個實例檢測您的 Cuties 是否成功安裝。

創建一個新的檔

從歡迎螢幕，或者從主功能表（Windows/Linux）/功能表列（Mac OS X）、工具列創建一個新的原始程式碼檔。如需瞭解更多有關創建檔的內容，請參閱創建新的代碼檔。

編寫你的代碼

在代碼編輯器中編寫你的代碼，這裡我們提供一個簡單的例子：

#include <stdio.h>
int main(int argc, char *argv[])
{
	printf("Hello World!\n");
	return 0;
}

這是一個非常簡單的C語言代碼，它的作用是在螢幕上顯示“Hello World!”。你可以複製或者鍵入這段代碼到 Cuties 之中。將其保存成 C 原始程式碼檔。如需瞭解更多有關保存原始程式碼檔的內容，請參閱保存你的代碼。

編譯和運行

你可以手動編譯這個原始程式碼然後手動運行。現在你還可以通過按一下工具列中的“編譯運行”按鈕直接完成這項操作。如需瞭解更多有關編譯運行的內容，請參閱構建你的程式。

現在螢幕上會出現 Cuties 的命令列運行程式（一個命令列視窗），上面顯示四行內容：第一行內容是“Hello World!”，第二行是一個空行。第三行顯示了程式的執行時間，游標處於第四行，提示您請按下鍵盤上的任意一個按鍵（Windows）/回車鍵（其他系統）來關閉命令列運行程式。
