User Interface(UI)

Based on rock-solid Qt foundation, Cuties provides you a incredible programming experience you never have. Start with the all new designed user interface.

Welcome Screen

When you start Cuties, it shows the Welcome Screen, where you can:

  • Create and open source file

  • Open recent source file

    You can click the item under "New Source File" to create a blank new file with different syntax highlighter.

    You can click the button "Open an Existing Source File" to open a source file on your local disk.

    You can also double click an item in the "Recent Opened Files" list to open a file you opened it recently in Cuties.

    Code Editor

    When you create or open a file, you will see the code editor. The tool bar will also appear. On the left, there are line number panel, break point mark panel and code folding marks. Those tools will help you to programming more easily.

    Sidebar

    Sidebar is folded at normal. When you move the mouse cursor on the left light slit of Cuties, the sidebar will appear. You can lock the sidebar in the menu.

    History Recording

    You can browse recent opened file in the sidebar. Click "History" button and sidebar will expand a recent opened file list.

    Clipboard

    Sidebar also provide a clipboard history list. Click "Clipboard" button and sidebar will show you a clipboard history. You will never have difficult to find out those code pieces you want to copy at once.

    Compiler

    Compiler dock will only appear when you need it. It will show all the errors and warnings in text view and tree view mode. You can double click the item to GOTO that line in tree view mode.

    Debugger

    When you start to debug your program, debugger controls docks will show up. Debug in Cuties will become super easy. From executing every command to every variable you watched.
