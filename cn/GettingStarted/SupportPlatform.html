支持的平台

Kreogist Cuties 允许您在多个系统平台上运行。

开发平台

Kreogist Cuties 为下列操作系统提供编译好的二进制安装包：

  • Microsoft® Windows®

  • Ubuntu Linux

  • Mac OS X

    参考设置

    一套设置是指对于某个平台来说，一套编译器、调试器和其他工具的搭配组合。下面是一些 Kreogist Dev Team 已进行过测试的设置，以供参考。

    操作系统平台编译器
    Ubuntu Linux 13.04, X11（64位）Ubuntu 提供的编译器
    Microsoft Windows 7（32位和64位）MinGW-builds gcc 4.8.1（32位）
    Microsoft Windows 8（32位和64位）MinGW-builds gcc 4.8.1（32位）
    Mac OS X 10.8 "Mountain Lion", Cocoa（64位）Apple 提供的 GCC 编译器

    您也可以使用您自己的设置编写代码。了解更多信息，请参阅编译器设置。

    可供使用的二进制安装包

    Cuties 为所支持的操作系统提供了二进制安装包。您可以从 http://kreogist.github.io/Cuties/cn/downloads.html 获得。此处提供了所有 Cuties 所支持操作系统的下载链接。
